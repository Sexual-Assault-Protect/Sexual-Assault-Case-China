## 电子科技大学张翼德事件


### 涉案人员说明

姓名：张翼德

性别：男

出生年月：1977年08月 

职称：副教授（丑闻前）

学位：博士 

属性：专职 

### 事件经过

2017.7 月初 .“女权之声”在微博发布信息呼吁大家关注电子科大的校园性骚扰事件。据多名答主在知乎爆料，电子科技大学通信学院张翼德老师曾在课堂上传播具有性别歧视和性骚扰意味的言论，并多次利用考试成绩、保研名额威逼利诱女生。

2017 年 7 月 6 日，电子科技大学通信抗干扰技术国家级重点实验室发布通告，暂停张翼德相关工作

2017 年 7 月底，电子科技大学发布通告，给予张翼德开除处分
